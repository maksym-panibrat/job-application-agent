# Chat-driven company suggestions (Layer 3) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a free-form `tags` field to each catalog row and expose the curated catalog (name + tags) to the onboarding/chat agent via a new `list_curated_companies` tool. The LLM does the ranking; suggestions land via the existing `save_profile_updates` path.

**Architecture:** One Postgres column (`companies.tags TEXT[]`), one Pydantic field (`CatalogRow.tags`), one SQLModel field (`Company.tags`), one seeder change (upsert sets `tags`), one read-only agent tool that returns curated rows as JSON, and a system-prompt paragraph telling the agent when to call it. Off-list resolution preserved via the existing `save_profile_updates` → `company_resolver.resolve` fan-out.

**Tech Stack:** Python 3.12, FastAPI + SQLModel + Alembic + Postgres, Pydantic v2 + PyYAML for the catalog parser, LangGraph + langchain_core for the chat agent. Tests: pytest + testcontainers + LangChain `FakeListChatModel`.

**Spec:** `docs/superpowers/specs/2026-05-08-chat-company-suggestions-design.md`.

**Prerequisites:** PR #107 (Layer 2 curated catalog) merged to `main`. The `Company.is_curated` column, `app/services/company_catalog.py` parser + seeder, `app/data/catalog/companies.yaml`, and `GET /api/companies/catalog` endpoint must all exist.

@tool
async def list_curated_companies(config: RunnableConfig) -> str:
    """Return the curated company catalog as a JSON array.
    Each entry has: canonical_name (str) and tags (list of strings).
    Call this when the user asks for company suggestions or to see
    what companies are available. Prefer suggesting from this list;
    you may suggest off-list names too — those get resolved against
    live ATS boards but won't have tags to reason over."""
    db_factory = config["configurable"]["db_factory"]
    async with db_factory() as session:
        rows = (
            await session.execute(
                select(Company.canonical_name, Company.tags)
                .where(Company.is_curated)
                .order_by(func.lower(Company.canonical_name))
            )
        ).all()
    payload = [
        {"canonical_name": r.canonical_name, "tags": list(r.tags)}
        for r in rows
    ]
    return json.dumps(payload)
